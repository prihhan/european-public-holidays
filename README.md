# European Public Holidays

A simple, elegant web application to view public holidays and school holidays from all European countries with an interactive calendar view and smart vacation planning suggestions.

![European Public Holidays](https://img.shields.io/badge/countries-30-blue) ![License](https://img.shields.io/badge/license-MIT-green) ![No Dependencies](https://img.shields.io/badge/dependencies-none-brightgreen)

## ✨ Features

- 🌍 **30 European Countries** - Complete holiday data for all major European nations
- 📅 **Interactive Calendar** - Visual calendar with color-coded holidays
- 🎒 **School Holidays** - Real-time school holiday data from OpenHolidays API
- 💡 **Bridge Day Suggestions** - Smart vacation planning with strategic day-off recommendations
- 🎨 **Beautiful Design** - Clean interface with country flags and smooth animations
- 📱 **Responsive** - Works perfectly on desktop, tablet, and mobile
- 🚀 **No Installation** - Runs directly in your browser

## 🎨 Color Coding

- 🔴 **Red**: Public holidays
- 🟡 **Yellow**: Shortened/optional days
- 🌸 **Light Pink**: School holidays

## 🚀 Quick Start

### Option 1: Open Locally
1. Download or clone this repository
2. Open `index.html` in your web browser
3. That's it! No build process or installation needed

### Option 2: Try it Online
Visit the live demo: `https://YOUR_USERNAME.github.io/european-public-holidays/`

## 📖 How to Use

1. **Select a Country** - Choose from the dropdown with flag icons
2. **Enter a Year** - Pick any year between 2020-2030
3. **Click "Load Holidays"** - View all holidays instantly
4. **Explore**:
   - See bridge day suggestions for optimal vacation planning
   - View school holidays (when available)
   - Click list header to expand/collapse holiday details
   - Hover over calendar dates for holiday information

## 💡 Bridge Day Examples

The app intelligently suggests strategic days to take off:
- "Take May 2 off and get a 4-day break" (when May 1 is a holiday on Thursday)
- "Take June 23 off and get a 5-day break" (when there's a holiday gap)
